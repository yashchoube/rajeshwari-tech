```bash

npm install

# Run all tests
npm run test:all

# Run specific test categories
npm run test:ui          # UI tests
npm run test:api         # API tests
npm run test:newsletter  # Newsletter tests
npm run test:integration # Integration tests
```

### **Test Execution**
```bash

cd tests
node run-tests.js

# Run individual test categories
```
